# Traffic Flow Prediction

This repository contains a traffic flow prediction system using machine learning techniques.

## Contents
- `Traffic_Flow_Prediction (1).ipynb` - Jupyter notebook with the implementation
- `traffic_dataset.mat` - MATLAB traffic dataset
- `ANN_22BAI10041_OVERLEAF_SOURCE` - Source files from Overleaf
- `ANN_22BAI10041_PDF.pdf` - PDF documentation
- `docs/` - Additional documentation

## About
A machine learning-based system for predicting traffic flow patterns. The implementation uses neural networks and other ML techniques to analyze and predict traffic conditions.
